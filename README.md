# To-Do-List

A to-do list is a written or digital list of tasks you plan to complete. Each item represents a specific action or goal, and the list helps you stay focused and organized.

# Working:-

- Add Tasks: Write down everything you need to do.
- Review & Update: Add new tasks or remove completed ones regularly.

# Helps in:-

- Reduces mental clutter
- Improves time management
- Boosts productivity and motivation
- Helps prioritize effectively
